Plots
=====

In this section, will be presented which plots you can use to visualize DREEM's output.

.. note:: 

   Most plots requires a Pandas dataframe, a (list of) tube(s), a (list of) construct(s) and/or a study.

.. note:: 

   A study is a series of tubes that are relevant to be studied together (replicates, change of only one experimental condition, etc).
